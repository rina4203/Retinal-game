**Genre:** Catcher + Rhythm
## About The Game

A 2D game featuring two visual themes (day and night) and two game modes (rhythm and endless).

## Game Modes

### Rhythm Mode 

* The player chooses a melody to play during the level.
* It features a system for "caught" and "missed" notes.
* If a certain number (N) of notes are missed, the level ends.
* 
### Endless Mode 

* The player does not choose a melody. 
* The falling speed of objects increases with every point caught.
* It features a system for "caught" and "missed" points.
* If a certain number (N) of points are missed, the game ends.

## Menu & Navigation

* Sound settings 
* Authors (Credits)
* Exit 
* Theme selection 
* Shop 

**Game Flow:**
Start -> Choose Mode -> Choose Difficulty 

## Controls 

* Arrow keys or A, D keys

## Extra Features 

* Score tracking and saving to set records.
* An in-game shop where you can spend in-game currency to buy different platform colors, etc.
